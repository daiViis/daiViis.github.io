🔧 Factory 84 – Update 001

Patch Notes

⚙️ Gameplay & Progression

Removed the following research upgrades:

Void Cooling
Counter-Intel
Ashen Reinforcement
Echo Theory

Updated win condition:
Players now win by delivering 1,000 Godcoins after building their first Portal.
This is presented as the final contract.

Automation now unlocks with the Printer License.
Automation panels remain hidden until this license is obtained.

Heat reaching 100% now triggers a Game Over with dedicated heat-related messages.

Added a Run Timer in the top-left corner.
The timer is saved and persists between sessions.

💰 Economy & Balance

Rebalanced building costs:
Cryo Pipe now costs 30 AI Cores.
Portal now costs 500 Timeline Ink.
Ash Cleaner has been significantly buffed:
Burnt Scrap removal increased from 0.25/s → 2/s.
Ash Cleaner upgrades now require Circuits instead of Scrap.
Storage upgrades now scale their material costs per level.

🖥️ Interface & Accessibility

Added UI text size options:
Smaller
Normal
Bigger

Removed the Upgrades tab.

Storage controls have been relocated under the grid.
Storage upgrades are now available via a new context menu.
Burnt Scrap Purge has been removed entirely.

Updated the main menu roadmap.

Added planned upgrades for the First Planet End Game.

📦 Logistics & Automation

Basic Logistics now unlocks the Buy Slot button.
This feature is only visible during the Basic Logistics contract.
Buy Slot has been moved under the grid in the support bar.
Buy Slot now uses the same cost chip styling as building cards and sits under Emergency.
Right-side automation panel and context automation remain hidden until unlocked.

🚨 Safety & Heat Management

Added an Emergency Shutdown button under the grid:
Powers off all buildings for 60 seconds.
Cooldown: 15 minutes.
Emergency button is now larger and stacked under the fan/storage counters.
[Bug Fix] Powered-off buildings no longer generate Burnt Scrap.
