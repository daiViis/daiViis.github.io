# Hellfactory

Static website for the Hellfactory project, hosted on GitHub Pages.

Live site: https://daiviis.github.io/

## Structure

- `index.html`: main menu entry
- `play.html`: game page
- `intro.html`: timeline intro
- `styles.css`: global styles
- `js/`: client-side scripts
- `images/`, `audio/`, `video/`: media assets
- `Updates/`: change notes

## Local use

Open `index.html` directly in a browser.

## Deploy

Push to `main` to publish via GitHub Pages.
